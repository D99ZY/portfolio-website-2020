Introduction

This is the application I developed for my final year project in University. The result was a fully functioning mobile app and a 50-page dissertation documenting the process including the preparation, design, development, and review of this project. Below is a brief summary of the project.




Proposal

This project aims to create a macro-nutrient and calorie tracking Android application with speech recognition that is faster and easier to use than existing health apps available today.




Design

The following high fidelity wireframe shows the main view of the MacroDiet app. From this page, the user can see all of the meals they have logged throughout the day, as well as each individual food item in every meal. Additionally, the quantity of each food and the associated nutritional information is displayed next to each food item. The accumulative nutritional values from each food item are calculated and displayed near the top of the page.

Designed in Adobe XD

Final Product

User Details Page

Add Food Page

Daily Log Page

The finished application allows users to input their details and nutrition goals when using the app for the first time. Then with each meal, the user can tell MacroDiet what foods they have eaten and how much of each food. The app will listen and then log the food item and the associated Calorie and macro-nutrient information based on the food amount. The user can then scroll through the Daily Log page to see exactly how much they have eaten to help keep in line with their health and nutrition goals.
